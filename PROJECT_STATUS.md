## 🔧 维护指南

### 添加新游戏
```python
# 使用游戏管理系统
python game_manager.py

# 或直接编辑数据
manager = GameManager()
manager.add_game({
    "title": "新游戏",
    "category": "puzzle",
    "url": "https://example.com/game",
    "description": "游戏描述"
})
```

### 更新游戏信息
```python
manager.update_game(game_id, {
    "rating": 4.8,
    "plays": 50000
})
```

### 验证游戏可用性
```python
manager.verify_game(game_id)
```
